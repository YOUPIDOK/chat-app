# Server
> Node 20
## Installation
**Create and config .env**
```shell
cp .env.example .env
```
**Install dependencies**
```bash
pnpm install
```
## Running server
**Development**
```bash
pnpm run start
```
**Watch mode**
```shell
pnpm run start:dev
```
**Production**
```shell
pnpm run start:prod
```
